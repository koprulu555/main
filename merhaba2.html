Kullanım Kılavuzu

Bu sayfa, API'den alınan token'i ham metin olarak döndürür. Aşağıdaki endpoint'e GET isteği yaparak token'i alabilirsiniz:

Endpoint: https://<your-domain>/token.html
Method: GET
Response: Ham token metni (text/plain)

Örnek kullanım (JavaScript):

fetch('https://<your-domain>/token.html')
.then(response => response.text())
.then(token => {
  // Token'i kullan
  console.log(token);
});
